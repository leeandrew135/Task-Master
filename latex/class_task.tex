\doxysubsection{Detailed Description}
Represents a task with properties such as name, date, status, priority, and description. 

The \doxylink{class_task}{Task} class provides methods to create, edit, view, and convert task information to JSON format. It supports basic getter and setter functionality for all relevant fields.
