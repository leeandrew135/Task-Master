Implementation of the \doxylink{class_todo_list}{Todo\+List} class. 

This file contains the implementation of the \doxylink{class_todo_list}{Todo\+List} class, which manages categorized task lists (Backlog, Doing, Review, Done).

The class allows for task creation, reading, status updates, deletion, and filtering by category or priority.

A helper function is also included to retrieve a task by ID from any category.
