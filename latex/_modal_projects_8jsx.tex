\doxysubsection{Detailed Description}
@component Modal\+Projects 

A modal that displays and manages user projects.

This component allows users to\+:
\begin{DoxyItemize}
\item View a list of their projects
\item Navigate to a project page
\item Delete a project
\item Create a new project and assign it to themselves
\end{DoxyItemize}

The modal toggles between selection and creation modes.
