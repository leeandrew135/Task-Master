\doxysubsection{Detailed Description}
@component Modal 

A modal form used to create a new task.

This component allows the user to input a task title, description, priority, and deadline. It converts the priority to a numeric value and passes all data to the {\ttfamily on\+Save} callback. Clicking outside the modal or pressing "{}\+Cancel"{} closes the form via {\ttfamily on\+Close}.

\end{DoxyParams}
