\doxysubsection{Detailed Description}
@component Sidebar 

A collapsible sidebar menu for navigation and logout.

This component overlays a sidebar with navigation buttons\+:
\begin{DoxyItemize}
\item Home\+: Navigates to /home
\item Calendar\+: Navigates to /calendar
\item Sign Out\+: Clears user data and returns to login screen
\end{DoxyItemize}

Clicking the dark background triggers {\ttfamily on\+Click} to close the sidebar.


\begin{DoxyParams}{Parameters}
{\em \{\+Function\}} & on\+Click -\/ Function called when the background is clicked (used to close the sidebar).\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
\{JSX.\+Element\} A modal-\/style sidebar for in-\/app navigation. 
\end{DoxyReturn}
