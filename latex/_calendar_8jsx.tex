\doxysubsection{Detailed Description}
@component Calendar 

A calendar component that displays the current month with user tasks on each day.

This component fetches a user\textquotesingle{}s tasks from the backend and displays them inside a calendar layout. Tasks are grouped by due dates, and the calendar includes navigation for previous and next months.

\begin{DoxyReturn}{Returns}
\{JSX.\+Element\} A full interactive calendar with tasks displayed on appropriate days.
\end{DoxyReturn}
