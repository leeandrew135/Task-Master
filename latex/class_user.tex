\doxysubsection{Detailed Description}
Represents a user with associated profile information and projects. 

The \doxylink{class_user}{User} class manages user-\/related data such as ID, name, email, and password, along with a collection of projects linked to the user. It provides functionality to add or remove projects and retrieve user details.
