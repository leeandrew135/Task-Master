\doxysubsection{Detailed Description}
@component Sign\+Up 

A modal component that handles user account registration.

This component allows new users to sign up by entering a username, email, password, and password confirmation. It checks for email uniqueness before submitting the new user data via the {\ttfamily on\+Save} callback.
