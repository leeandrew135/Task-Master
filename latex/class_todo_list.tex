\doxysubsection{Detailed Description}
Manages a categorized collection of tasks using a Kanban-\/style system. 

The \doxylink{class_todo_list}{Todo\+List} class organizes tasks into four categories\+: backlog, doing, review, and done. It supports creating, reading, updating, deleting, and filtering tasks based on category or priority. Internally, each category is stored as a separate vector of \doxylink{class_task}{Task} objects.

This class uses helper methods for task lookup and supports status transitions.
