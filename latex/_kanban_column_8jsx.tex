\doxysubsection{Detailed Description}
@component Kanban\+Column 

A droppable column in the Kanban board that holds draggable tasks.

Each column represents a task status (e.\+g., Backlog, Doing, Review, Done), and is configured as a droppable area using {\ttfamily @dnd-\/kit}. Tasks passed in via props are rendered inside the column as draggable cards.


\begin{DoxyParams}{Parameters}
{\em \{string\}} & title -\/ The title of the column (used as droppable ID). \\
\hline
{\em \{\+Array\}} & tasks -\/ An array of task objects to render in this column.\\
\hline
\end{DoxyParams}
